The project seeks to apply foundational statistical methods to uncover meaningful patterns, associations, and predictive relationships within the mtcars dataset.
The main goals are:
To understand the basic properties of key variables (horsepower and miles per gallon) by calculating measures of central tendency (mean), variability (standard deviation), and distribution shape
To investigate the linear relationship between horsepower and fuel efficiency using Pearsonâ€™s correlation, which helps identify how changes in engine power relate to changes in miles per gallon.
To develop a simple linear regression model predicting fuel efficiency from horsepower, providing insights on how these two variables interact and quantifying the strength and direction of their relationship.
To visually illustrate the relationship between horsepower and fuel efficiency by creating a scatter plot complemented with a regression (trend) line, making it easier to interpret the results and observe patterns.
